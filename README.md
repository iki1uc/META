META — AX/IS/XA‑Dreifach‑Modul der iki1uc‑Public‑Ebene
META ist ein dreiachsiges Public‑Meta‑Modul innerhalb der iki1uc‑Struktur.
Es dient der Darstellung von Vorwärts‑Frames (AX), Zustands‑Frames (IS) und Rückführungs‑Frames (XA).
META ist vollständig neutral, konfliktfrei und MARKT‑3‑kompatibel.

META arbeitet ausschließlich in der Public‑Ebene und trägt keine System‑Last.

🟦 Öffentlicher Zweck
META stellt die Public‑Meta‑Schicht bereit:

AX — Vorwärts‑Achse (forward.js)

IS — Zustands‑Achse (state.js)

XA — Rückführungs‑Achse (return.js)
